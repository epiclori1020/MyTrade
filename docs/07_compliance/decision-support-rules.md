# Decision Support Regeln — MyTrade

## Grundsatz
Das System bietet **Decision Support, keine Anlageberatung.**
Es sammelt Daten, analysiert, verifiziert und empfiehlt — der Mensch entscheidet und handelt.

## Regeln

1. **Keine Haftung:** System-Outputs sind informativ, nicht verbindlich.
2. **Human-in-the-Loop:** In Stufe 1-2 braucht jeder Trade menschliche Bestätigung.
3. **Disclaimer:** Jeder Trade-Plan enthält den Hinweis: "Dies ist keine Anlageberatung."
4. **Keine Garantien:** System macht keine Performance-Versprechen.
5. **Transparenz:** Jede Empfehlung zeigt die zugrunde liegenden Daten, Quellen und Verification-Status.
6. **Audit Trail:** Alle Entscheidungen (System + User) werden in Supabase geloggt.
7. **Widerspruch möglich:** User kann jede Empfehlung ablehnen ohne Konsequenz.

## Was das System NICHT darf
- Sich als Finanzberater ausgeben
- Steuerliche Empfehlungen geben (nur Fakten darstellen)
- Dringlichkeit suggerieren ("Sie müssen jetzt kaufen!")
- Performance-Prognosen als Fakten darstellen
- Andere Nutzer beraten (Single-User-System)
